This repository contains the implementation for the "Dynamic scenes" project.

## Prerequisites

To run the project, ensure you have Python installed along with the following libraries:


- `vvrpywork` (Course visualization framework)
- `numpy`
- `laspy` (for reading LIDAR data)
- `lazrs` (decompression backend for `.laz` files)

You can install the required Python packages using:
```bash
pip install numpy laspy lazrs
```
